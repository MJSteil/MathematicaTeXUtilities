TeXCommand

TeXCommand["name"]
represents TeX command \name .
TeXCommand["name", {arg1, arg2, ...}]
represents TeX command with arguments \name {arg1}{arg2}....
TeXCommand["name", {{opt1→val1, opt2, opt3→val3, ...}, arg1, arg2, ...}]
represents TeX command with optional arguments \name [opt1=val1,opt2,opt3=val3]{arg1}{arg2}....
MORE INFORMATION
  • The following options can be given:
"ArgumentConverter"ToString[#1, TeXForm] & function used to convert command arguments, optional argument names and values, to . This function should accept one argument and return a string
